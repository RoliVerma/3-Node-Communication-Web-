GUIDELINES

This website is for the registered libraries and hostels of KIIT University Bhubneswar.

A list of students who generate the pass for the day is made available to the respective library and the hostel of the student.

The librarian is expected to generate new QR code for the student to get verified as present as soon as he/she reaches the library.

The hostel is expected to generate new QR code for the student to get verified as returned back as soon as he/she reaches the hostel.

Both the hostel and library are expected to delete the database before a month ends.They will be getting a notification for a same thrice in a month.
